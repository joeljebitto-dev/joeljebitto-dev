## Featured Projects

### Franka Panda MuJoCo Exp-2

A 7-DOF Franka Panda robot simulation framework built with Python, MuJoCo, FastAPI, and React.

Key areas:

- 7-DOF Franka Panda robot wrapper
- MuJoCo viewer and headless simulation
- Robot state snapshots
- MuJoCo-based forward kinematics and Jacobian helpers
- Damped least-squares inverse kinematics
- Joint-space PID control
- Linear, cubic, and quintic joint trajectory generation
- FastAPI + React real-time dashboard
- Tests for PID, trajectory, kinematics, and simulation behavior

Repository: [Franka_Panda_Mujoco_Exp-2](https://github.com/joeljebitto-dev/Franka_Panda_Mujoco_Exp-2)

## Engineering Mindset

I try to build with these principles:

- Understand the fundamentals before adding complexity
- Build small working systems first, then improve them
- Keep code readable and modular
- Separate UI, backend, simulation, control, and math logic clearly
- Write projects that are easy to run and explain
- Use tests where possible
- Document setup, usage, project layout, and future improvements
- Connect software decisions to real engineering constraints
